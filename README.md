# SecurityLib

**SecurityLib** is a lightweight and reusable C# library providing essential security utilities for .NET applications.  
It simplifies common security tasks such as authentication, encryption, hashing, and secure storage, helping developers follow best practices with minimal boilerplate.

---

## Features

- **Authentication Helpers**: Simplify token generation, validation, and user authentication flows.
- **Encryption & Decryption**: AES, RSA, or other supported algorithms for securing sensitive data.
- **Hashing Utilities**: Password hashing with salt, secure comparison, and integrity checks.
- **Configuration & Secrets Management**: Easy integration with app settings and user secrets for sensitive information.
- **Extensible & Reusable**: Modular structure allows integration into any .NET project without heavy dependencies.
- **Security Best Practices**: Encapsulates common security patterns to reduce implementation errors.
